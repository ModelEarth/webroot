### Start Local Server

From the webroot folder, run:
python -m http.server 8887


Then view at:

[localhost:8887](http://localhost:8887)  
[localhost:8887/team](http://localhost:8887/team/)  
[localhost:8887/projects](http://localhost:8887/projects/)
