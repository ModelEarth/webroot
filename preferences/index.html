Development Note: Configuration for GitHub Pages Deployments

For OAuth authentication with GitHub Pages hosting, you'll need to:

  • Store OAuth credentials as GitHub Secrets in your repository
  • Use GitHub Actions to inject credentials during build time
  • Consider using environment variables like DECIDIM_CLIENT_ID
  • Implement proper CORS handling for your redirect URIs

OAuth redirect URLs should point to your GitHub Pages domain (e.g., https://yourusername.github.io/repo-name/)
