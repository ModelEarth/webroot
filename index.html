Webroot setup


1. Install Github Desktop
2. Fork the webroot repo
3. Click on the green "Code" button
4. Then "Open with GitHub Desktop"
5. Clone the repo
6. Choose "To contribute to the parent project"
7. In your local file manager, right-click on "webroot" and choose "New Terminal at Folder",
